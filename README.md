# 🎵 Spotify Web Player Clone

A Spotify-inspired music streaming web player built using HTML and CSS. The project recreates Spotify's modern interface with a responsive layout, music sections, and player controls.

## ✨ Features

- 🏠 Navigation & Library
- 🎵 Music Cards 
- 🎧 Music Player Controls
- 🔊 Volume Slider
- 📱 Responsive UI
- 🎨 Spotify-Inspired Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome
- Google Fonts (Montserrat)
